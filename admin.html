How updates reach customers
This page is for you only. After you change prices or photos, click Download products.json and upload that file to your GitHub repository (replace the old products.json), then push — then everyone sees the new catalogue.

Save in this browser only previews changes on this phone or computer; it does not change the live website for friends.

Security: Username and password in this page are not secret (anyone can view the code). Use it for convenience only, not for private data.

Bookmark this page as: admin.html (do not share the link publicly if you want casual visitors to stay out).
